# kuchiku

1) note
This is a Perl script which show you the list of the Destroyes of Imperial Japanese Navy.


2) usage

./kuchiku -option

for exp
./kuchiku -fubuki


3) help option

following command show you all options
./kuchiku -help

-mutsuki
-fubuki
-hatsuharu
-shiratsuyu
-asashio
-kagerou
-yuugumo
-akizuki
